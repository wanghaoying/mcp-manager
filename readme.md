# MCP 市场管理平台

## 项目概述

本项目基于 Golang 1.23 开发，旨在构建一个 MCP（Model Context Protocol）工具市场管理平台，实现以下核心功能：

### 1. Swagger 文档导入与转换
- 支持导入符合 OpenAPI 规范的 Swagger 文档
- 将文档解析转换为可通过 MCP 协议暴露的工具

### 2. MCP 服务集成
- 内置 MCP-Server 用于工具注册与管理
- 支持多种暴露方式：SSE（Server-Sent Events）、标准输入输出（stdio）等
